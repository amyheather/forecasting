# Forecasting: Theory and Research Notes

This repository contains:

* **Theory:** Beginner friendly introduction to forecasting, including ARIMA and Prophet. Draws heavily on a range of existing resources (cited within).

* **Research notes:** Describes existing research with the South Western Ambulance Service NHS Foundation Trust (SWASFT) and plans for ongoing work.

View at: https://ambmodels.github.io/forecasting-research/

## Viewing locally

Install the provided uv environment then run:

```
uv run quarto preview website
```
